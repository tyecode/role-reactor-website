1. Community Health (The Basics)
   Total Members: (Universal) usually split into Humans vs. Bots.
   Membership Growth:
   Net Growth: +5 members today.
   Joins vs. Leaves: A simple line graph showing user flow over 7/30 days.
   Online Count: How many users are currently online (indicates server "aliveness").
   Server Boosts: Boost level and total boost count.
2. Activity & Engagement (Gamification)
   Most Active Members:
   XP Leaderboard: Top 3-5 users by XP/Level (common in MEE6/ProBot).
   Message Count: Top chatters.
   Activity Graphs:
   Messages Sent: Line chart of total messages per day.
   Voice Minutes: Total time users spent in voice channels.
   Channel Activity: Which channels are most active? (e.g., #general vs #memes).
3. Feature Utilization (Bot Specific)
   Command Usage: How many times users used bot commands (e.g., /rank, /play).
   Module Status: Which plugins are active? (Moderation, Welcomer, Music).
   Recent Actions:
   Moderation Logs: Last 5 bans/kicks/mutes (Dyno/Carl-bot heavily focus on this).
   Welcome Logs: List of recently joined users.
4. Technical / Server Stats
   Counts: Roles, Channels, Emojis.
   Bot Latency: API Ping (ms).
   Uptime: Bot uptime percentage.
