## Usage Examples

<Tabs items={["Create", "Apply", "Manage"]}>
  <Tab value="Create">
    Create a role bundle:
    ```bash
    /role-bundle create name:VIP roles:@VIP,@Supporter,@Donator
    ```
    Groups multiple roles under one reusable name.
  </Tab>
  <Tab value="Apply">
    Apply a bundle to a user:
    ```bash
    /role-bundle apply name:VIP user:@member
    ```
    All roles in the bundle are assigned at once.
  </Tab>
  <Tab value="Manage">
    List or delete bundles:
    ```bash
    /role-bundle list
    /role-bundle delete name:VIP
    ```
  </Tab>
</Tabs>

## Permissions Required

<Callout type="info">
  - **Manage Roles** permission required
</Callout>

## What You'll See

<Callout type="info">
  Confirmation of bundle creation, deletion, or application. When applying a
  bundle, the bot assigns all included roles to the target user in one action.
</Callout>

## Troubleshooting

<Callout type="warning">
  - **Role not assigned:** The bot's highest role must be above the roles in the
  bundle.
  <br />- **Bundle not found:** Use `/role-bundle list` to see all available
  bundles.
  <br />- **Cannot create:** Make sure you have Manage Roles permission.
</Callout>
