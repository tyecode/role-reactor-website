## Usage Examples

<Tabs items={["Timeout", "Warn", "Ban/Kick", "Purge"]}>
  <Tab value="Timeout">
    Timeout a user:
    ```bash
    /moderation timeout users:@user duration:1h reason:Spamming
    ```
    - Supports multiple users separated by commas
    - Duration format: `10s`, `5m`, `1h`, `1d`, `1w` (max 28 days)
  </Tab>
  <Tab value="Warn">
    Warn a user:
    ```bash
    /moderation warn users:@user reason:Breaking rules
    ```
    Warnings are logged in the user's moderation history.
  </Tab>
  <Tab value="Ban/Kick">
    Ban or kick users:
    ```bash
    /moderation ban users:@user reason:Repeated violations
    /moderation kick users:@user reason:Inappropriate behavior
    ```
  </Tab>
  <Tab value="Purge">
    Bulk delete messages:
    ```bash
    /moderation purge amount:50
    /moderation purge amount:100 user:@spammer
    ```
  </Tab>
</Tabs>

## Permissions Required

<Callout type="info">
  - **Moderate Members** — Required for timeout
  - **Ban Members** — Required for ban
  - **Kick Members** — Required for kick
  - **Manage Messages** — Required for purge
</Callout>

## What You'll See

<Callout type="info">
  Confirmation embeds for each moderation action with details about who was
  moderated, the reason, and duration. Actions are logged in the user's
  moderation history for future reference.
</Callout>

## Troubleshooting

<Callout type="warning">
  - **Cannot moderate user:** You can only moderate users with a lower role than
  yours. Server owners and bots cannot be moderated.
  <br />- **Duration too long:** Timeouts have a maximum of 28 days.
  <br />- **Purge limit:** Discord only allows purging messages less than 14
  days old.
</Callout>
