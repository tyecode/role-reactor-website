## Usage Examples

<Tabs items={["Basic Setup", "Manage Settings"]}>
  <Tab value="Basic Setup">
    Configure the welcome system:
    ```bash
    /welcome setup channel:#welcome message:Welcome {user} to {server}! 🎉
    ```
    Optional settings:
    - `auto-role` - A role to assign to new members automatically
    - `enabled` - Toggle the system on/off (default: true)
    - `embed` - Send the message as an embed instead of text (default: false)
  </Tab>
  <Tab value="Manage Settings">
    View and toggle settings:
    ```bash
    /welcome settings
    ```
    Opens an interactive dashboard to test your welcome message, toggle features, and update the configuration.
  </Tab>
</Tabs>

## Placeholders

You can use these placeholders in your welcome message:

- `{user}` - User mention
- `{username}` - Username without mention
- `{server}` - Server name
- `{memberCount}` - Total member count

## Troubleshooting

<Callout type="warning">
  - **Permission denied:** You require Manage Server permissions to configure the
  welcome system.
  <br />
  - **Channel not found:** Verify the channel exists and the bot can access it.
  <br />
  - **Auto-role not working:** Ensure the bot's role is positioned above the
  auto-role in the server settings hierarchy.
  <br />
  - **Message not sending:** Verify bot permissions in the welcome
  channel.
</Callout>
