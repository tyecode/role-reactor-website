## Overview

The Stats command provides a comprehensive overview of the bot's performance and usage across all servers it serves.

## What You'll See

<Callout type="info">
  - **Server Count** — Total number of servers the bot is in - **Member Count**
  — Total members across all servers - **Command Usage** — Most frequently used
  commands - **Uptime** — How long the bot has been running
</Callout>

## Usage

```bash
/stats
```

No additional parameters required.
