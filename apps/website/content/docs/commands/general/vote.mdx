## Step-by-Step Example

1. Run the command:
   ```bash
   /vote
   ```
2. The bot shows your voting status with:
   - **Vote Reward** — 1 Core per vote
   - **Cooldown Status** — whether you can vote now or when it resets
   - **Your Total Votes** — lifetime vote count
   - **How It Works** — step-by-step guide
3. Click the **Vote on Top.gg** button to open the voting page.
4. Log in with Discord, click **Vote**, and you'll automatically receive **1 Core**.
5. Come back in **12 hours** to vote again!

## Troubleshooting

<Callout type="warning">
  - **Didn't receive reward:** Make sure you voted while logged into the correct
  Discord account. Rewards are delivered automatically.
  <br />- **Cooldown active:** You can only vote once every 12 hours. The
  command shows when you can vote again.
</Callout>
