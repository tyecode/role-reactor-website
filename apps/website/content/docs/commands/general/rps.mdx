## Step-by-Step Example

1. Run the command:
   ```bash
   /rps user:@friend choice:Rock
   ```
2. Your choice is hidden from the opponent.
3. If you challenged a **bot**, the result is shown immediately.
4. If you challenged a **user**, they'll see interactive buttons to pick their choice.
5. Once both players have chosen, the bot reveals the result!

## Game Rules

- **Rock** beats Scissors
- **Paper** beats Rock
- **Scissors** beats Paper
- Same choice results in a **tie**

## Troubleshooting

<Callout type="warning">
  - **Opponent didn't respond:** The challenge expires after a short time. Ask
  them to try again!
  <br />- **Can't challenge yourself:** You cannot play against yourself — pick
  another user or challenge the bot.
</Callout>
