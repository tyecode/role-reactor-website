## Step-by-Step Example

1. View your own info:
   ```bash
   /userinfo
   ```
2. Or view another user's info:
   ```bash
   /userinfo user:@username
   ```
3. The bot displays a detailed embed with:
   - Username, avatar, and display name
   - Account creation date
   - Server join date
   - Assigned roles
   - Current activity and status
   - User badges and flags

## Permissions Required

<Callout type="info">
  - **View Channel** permission required
</Callout>

## What You'll See

<Callout type="info">
  A detailed embed with comprehensive user information including account
  creation date, server join date, roles, permissions, and current activity.
  Great for checking user details and account information.
</Callout>

## Troubleshooting

<Callout type="warning">
  - **User not found:** Make sure you're mentioning a valid user who is in the
  server.
  <br />- **Missing roles:** The bot can only show roles it has permission to
  see.
</Callout>
