## Step-by-Step Example

1. Get a random question:
   ```bash
   /wyr
   ```
2. Or pick a specific category:
   ```bash
   /wyr category:Funny
   ```
3. The bot posts a "Would You Rather" question for the channel to debate!

## Troubleshooting

<Callout type="warning">
  - **Same question again:** Questions are random, so duplicates can happen.
  Try a different category for variety!
  <br />- **No response:** Make sure the bot is online and has permission to
  send messages.
</Callout>
