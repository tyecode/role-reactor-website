## Complete Command Reference

### General Commands

- `/help` - Access help and information about bot commands
- `/ping` - Verify bot latency and connection status
- `/invite` - Retrieve the bot invite link
- `/support` - Access the support server link
- `/avatar` - Generate AI-powered avatars with custom prompts
- `/8ball` - Ask the magic 8ball a question
- `/core balance` - View your current Core balance and tier status
- `/poll create` - Create a new poll using interactive form
- `/poll list` - List all polls in the server
- `/poll end` - End an active poll early
- `/poll delete` - Delete a poll permanently
- `/level` - View your level and experience
- `/leaderboard` - View the server experience leaderboard
- `/rps` - Challenge someone to Rock Paper Scissors
- `/serverinfo` - Display detailed information about the server
- `/userinfo` - Display detailed information about a user
- `/vote` - Vote for the bot on top.gg and earn rewards
- `/wyr` - Get a random "Would You Rather" question

### Admin Commands

- `/role-reactions` - Manage role reaction messages
- `/temp-roles` - Manage temporary role assignments
- `/schedule-role` - Schedule role assignments
- `/welcome` - Configure the welcome system and settings
- `/goodbye` - Configure the goodbye system and settings
- `/xp` - Manage the XP system, settings, and level rewards
- `/giveaway` - Create and manage server giveaways
- `/moderation` - Timeout, warn, ban, kick, and purge members
- `/role-bundle` - Manage reusable role bundles
- `/ticket` - Manage the ticket support system
- `/voice-roles` - Auto-control users in voice based on roles

## Getting Started

### For New Users

1. **Begin with `/help`** - View all available commands and their descriptions
2. **Test basic commands** - Execute `/ping` to verify bot functionality
3. **Explore features** - Execute `/level` to view your experience points
4. **Access support** - Execute `/support` if you require assistance

### For Server Administrators

1. **Configure roles** - Execute `/role-reactions setup` to create role selection messages
2. **Configure welcome** - Execute `/welcome` to welcome new members
3. **Configure goodbye** - Execute `/goodbye` to send goodbye messages
4. **Manage XP** - Execute `/xp` to configure the experience system
5. **Schedule roles** - Execute `/schedule-role` for automated role assignments
6. **Set up tickets** - Execute `/ticket setup` to create a support system
7. **Configure moderation** - Use `/moderation` for timeout, warn, ban, and kick

## Common Use Cases

### Setting Up a New Server

1. **Welcome new members:** Execute `/welcome` to configure automatic greetings
2. **Goodbye messages:** Execute `/goodbye` to configure goodbye messages
3. **Create role selection:** Execute `/role-reactions setup` for self-assignable roles
4. **Enable XP system:** Execute `/xp` to configure experience points
5. **Set up tickets:** Execute `/ticket setup` for organized support
6. **Verify functionality:** Execute `/ping` and `/help` to confirm the bot is operational

### Managing an Active Server

1. **Monitor activity:** Execute `/leaderboard` to view most active members
2. **Assign temporary roles:** Execute `/temp-roles assign` for events or moderation
3. **Schedule role changes:** Execute `/schedule-role` for automated role assignments
4. **Moderate members:** Execute `/moderation` for timeout, warn, ban, or kick
5. **Run giveaways:** Execute `/giveaway create` for community events
6. **Voice control:** Execute `/voice-roles` to manage voice channel behavior
