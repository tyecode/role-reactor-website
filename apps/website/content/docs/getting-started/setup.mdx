## Quick Start

<Callout type="info">
  **Formatting Note:** When creating role strings, avoid extra spaces around
  commas and ensure each emoji is unique. Use `🎮:Gaming,🎨:Art` not `🎮:Gaming,
  🎨:Art`.
</Callout>

<Steps>
  <Step>
    **Invite the Bot**
    
    [Add Role Reactor Bot to your server](https://discord.com/api/oauth2/authorize?client_id=1392714201558159431&scope=bot+applications.commands&permissions=268790880)
    
    Ensure the following permissions are granted:
    
    Manage Roles, Manage Messages, Send Messages, Add Reactions, Read Message History, View Channel, Embed Links, Manage Server, Use External Emojis

  </Step>

<Step>
  **Create Your First Role Message**
  
  ```bash
  /role-reactions setup title:Server Roles description:Choose your roles below! roles:🎮:Gaming,📚:Books,🎵:Music
  ```

</Step>

  <Step>
    **Verify Functionality**
    
    Test the role message by clicking reactions to confirm roles are assigned and removed correctly.
  </Step>
</Steps>

<Callout type="success">
  Setup complete. You now have a functional role system. Continue reading for
  advanced configuration options.
</Callout>

### Permission Configuration

1. **During Invitation**: Select all required permissions when inviting the bot
2. **Post-Installation**: Verify bot permissions in Server Settings > Roles
3. **Role Position**: Ensure the bot's role is positioned above any roles it manages

<Callout type="warn">
  **Important**: The bot can only manage roles that are positioned below its own
  role in the server hierarchy.
</Callout>

## Creating Role Messages

### Basic Role Message

<Tabs items={["Simple Format", "With Categories", "Advanced Options"]}>
  <Tab value="Simple Format">
    **Basic role list:** ``` /role-reactions setup title:Choose Your Roles
    description:React to get roles! roles:🎮:Gaming,📚:Reading,🎵:Music,🎨:Art
    ``` **Output:** ``` Choose Your Roles React to get roles! 🎮 📚 🎵 🎨 ```
  </Tab>

<Tab value="With Categories">
  **Organized with categories:** ```bash /role-reactions setup title:Game Roles
  description:Choose your favorite games! roles:🔴:Valorant,⚡:Apex
  Legends,⚔️:League of Legends ``` **Output:** ``` Game Roles Choose your
  favorite games! 🔴 ⚡ ⚔️ ```
</Tab>

  <Tab value="Advanced Options">
    **Full customization:** ```bash /role-reactions setup title:🌟 Community
    Roles description:**Welcome!** Choose your roles below.\n\n*You can select
    multiple roles.* roles:#Gaming 🎮\n🖥️:PC
    Gaming,🎮:Console,📱:Mobile\n#Creative 🎨\n🎨:Art,📸:Photo,🎵:Music ```
  </Tab>
</Tabs>

### Category Organization

Organize roles into categories for improved user experience:

<Steps>
  <Step>
    **Category Header** Begin with `#Category Name` to create a section header
  </Step>

<Step>
  **Add Roles** List roles under each category: `emoji:role1,emoji:role2`
</Step>

  <Step>
    **Multiple Categories** Separate categories with `\n` for new lines
  </Step>
</Steps>

**Example Structure:**

```
#Gaming Platforms
🖥️:PC Gaming,🎮:Console Gaming,📱:Mobile Gaming
#Notifications
📢:Announcements,🎉:Events,📰:News
#Interests
🎨:Art,📚:Books,🎵:Music
```

### Role Color Configuration

**Setting Role Colors:**

Roles are created through Discord's server settings. The bot can only manage existing roles.

1. Navigate to Server Settings > Roles
2. Create new roles with your desired colors
3. Use the bot to assign these roles via reactions

**Color Guidelines:**

- Use hex color codes (#FF0000) for precise control
- Group related roles in similar color families
- Ensure sufficient contrast for readability
- Consider colorblind accessibility

### Role Channel Best Practices

<Callout type="info">
  **Recommendation:** Create a dedicated #get-roles channel for optimal user
  experience.
</Callout>

**Channel Configuration:**

1. Create a channel named `#get-roles` or `#roles`
2. Configure permissions (everyone can view, limited sending)
3. Pin important role messages
4. Add channel description explaining reaction usage

**Channel Description Example:**

```
🎭 Role Selection

React to the messages below to get roles!
• Click an emoji to get that role
• Click again to remove the role
• You can have multiple roles
• Ask in #general if you need help
```

## Advanced Configuration

### Role Hierarchy Management

**Important Rules:**

- Bot's role must be positioned above managed roles
- Higher roles in the list have more permissions
- Exercise caution with administrative roles

**Hierarchy Setup:**

1. Navigate to Server Settings > Roles
2. Drag the bot's role above any roles it should manage
3. Arrange other roles by importance
4. Test role assignment to verify functionality

## Troubleshooting

### Common Issues

<Steps>
  <Step>
    **Bot Not Responding** - Verify bot permissions - Check bot role position -
    Confirm bot is online
  </Step>

<Step>
  **Roles Not Being Assigned** - Verify role hierarchy (bot role above target
  roles) - Confirm role names match exactly - Test with a simple role first
</Step>

  <Step>
    **Reactions Not Working** - Contact support to fix reactions - Verify emojis
    are from the server or Unicode - Confirm bot can add reactions in the
    channel
  </Step>
</Steps>
