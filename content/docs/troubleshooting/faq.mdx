<Tabs items={["General", "Roles", "Permissions", "Commands"]}>
  <Tab value="General">
    **Q: Is Role Reactor free to use?**
    
    A: Yes, Role Reactor is free for all Discord servers.
    
    **Q: Where can I get support?**
    
    A: Join our support server or check the documentation.
  </Tab>
  <Tab value="Roles">
    **Q: Why aren't roles being assigned?**
    
    A: Check the bot's permissions and role hierarchy. See the troubleshooting guide for details.
    
    **Q: Can I use custom emoji?**
    
    A: Yes, but the bot must have access to the emoji.
  </Tab>
  <Tab value="Permissions">
    **Q: What permissions does the bot need?**
    
    A: Manage Roles, Send Messages, Add Reactions, Use Slash Commands.
    
    **Q: How do I fix missing permissions?**
    
    A: Update the bot's role in Server Settings > Roles.
  </Tab>
  <Tab value="Commands">
    **Q: Why do I get 'Missing Permissions' errors?**
    
    A: You need the correct Discord permissions to use certain commands. See the command docs for details.
    
    **Q: How do I update a role message?**
    
    A: Use `/role-reactions update` with the message ID from `/role-reactions list`.
  </Tab>
</Tabs>
