## Complete Command Reference

### General Commands

- `/help` - Access help and information about bot commands
- `/ping` - Verify bot latency and connection status
- `/invite` - Retrieve the bot invite link
- `/support` - Access the support server link
- `/sponsor` - View information about sponsoring the bot
- `/avatar` - Generate AI-powered avatars with custom prompts
- `/8ball` - Ask the magic 8ball a question
- `/core balance` - View your current Core balance and tier status
- `/core pricing` - View Core pricing and membership benefits
- `/poll create` - Create a new poll using interactive form
- `/poll list` - List all polls in the server
- `/poll end` - End an active poll early
- `/poll delete` - Delete a poll permanently
- `/level` - View your level and experience
- `/leaderboard` - View the server experience leaderboard

### Admin Commands

- `/role-reactions` - Manage role reaction messages
- `/temp-roles` - Manage temporary role assignments
- `/schedule-role` - Schedule role assignments
- `/welcome setup` - Configure the welcome system
- `/welcome settings` - View and manage welcome system settings
- `/goodbye setup` - Configure the goodbye system
- `/goodbye settings` - View and manage goodbye system settings
- `/xp settings` - View and manage XP system settings

## Getting Started

### For New Users

1. **Begin with `/help`** - View all available commands and their descriptions
2. **Test basic commands** - Execute `/ping` to verify bot functionality
3. **Explore features** - Execute `/level` to view your experience points
4. **Access support** - Execute `/support` if you require assistance

### For Server Administrators

1. **Configure roles** - Execute `/role-reactions setup` to create role selection messages
2. **Configure welcome** - Execute `/welcome setup` to welcome new members
3. **Configure goodbye** - Execute `/goodbye setup` to send goodbye messages
4. **Manage XP** - Execute `/xp settings` to configure the experience system
5. **Schedule roles** - Execute `/schedule-role` for automated role assignments

## Common Use Cases

### Setting Up a New Server

1. **Welcome new members:** Execute `/welcome setup` to configure automatic greetings
2. **Goodbye messages:** Execute `/goodbye setup` to configure goodbye messages
3. **Create role selection:** Execute `/role-reactions setup` for self-assignable roles
4. **Enable XP system:** Execute `/xp settings` to configure experience points
5. **Verify functionality:** Execute `/ping` and `/help` to confirm the bot is operational

### Managing an Active Server

1. **Monitor activity:** Execute `/leaderboard` to view most active members
2. **Assign temporary roles:** Execute `/temp-roles assign` for events or moderation
3. **Schedule role changes:** Execute `/schedule-role` for automated role assignments
4. **View user information:** Execute `/level` and `/avatar` for member details
