## Step-by-Step Example
1. Execute the command:
   ```bash
   /invite
   ```
2. The bot will send an ephemeral message (only visible to you) with an invite link to add Role Reactor Bot to your server.
3. Share the link with others to invite the bot.

## Permissions Required
<Callout type="info">
- **No special permissions required** - Available to all users
</Callout>

## What You'll See
<Callout type="info">
An invite link to add Role Reactor Bot to your server. The link is sent as an ephemeral message, so only you can see it. Share it with others to invite the bot!
</Callout>

## Troubleshooting
<Callout type="warning">
- **No response:** Verify the bot is online and can send messages.<br />
- **Invite link expired:** Contact the bot developer for a new invite.
</Callout>
