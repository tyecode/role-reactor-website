## Step-by-Step Example
1. Run the command:
    ```bash
    /invite
    ```
2. The bot will reply with an invite link.

## Troubleshooting
<Callout type="warning">
- **No response:** Make sure the bot is online and can send messages.<br />
- **Invite link expired:** Contact the bot developer for a new invite.
</Callout>
