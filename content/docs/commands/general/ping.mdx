## Step-by-Step Example
1. Run the command:
   ```bash
   /ping
   ```
2. The bot will respond with:
   - Bot latency (response time)
   - API latency (Discord API response time)
   - Uptime information
   - Connection status

## What the Response Shows
- **Bot Latency:** How fast the bot processes commands
- **API Latency:** Discord API response time
- **Uptime:** How long the bot has been running
- **Status:** Whether the bot is functioning properly

## Troubleshooting
<Callout type="warning">
- **High latency:** The bot may be experiencing performance issues.<br />
- **No response:** The bot may be offline or experiencing problems.<br />
- **API errors:** Discord API may be experiencing issues.
</Callout>
