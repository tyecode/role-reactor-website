## Step-by-Step Example
1. Access general help:
   ```bash
   /help
   ```
2. Access detailed help for a specific command:
   ```bash
   /help command:role-reactions
   ```
3. The bot will display comprehensive instructions, examples, and detailed information about using the bot effectively.

## Troubleshooting
<Callout type="warning">
- **No response:** Verify the bot is online and has permission to send messages.<br />
- **Command not found:** Verify the command name when using the optional command parameter.
</Callout>
