## Step-by-Step Example
1. To check your own level:
   ```bash
   /level
   ```
2. To check someone else's level:
   ```bash
   /level user:@username
   ```
3. The bot will show detailed level information including current XP, progress to next level, and rank.

## Features
- **Current level:** Shows the user's current level
- **Experience points:** Displays total XP earned
- **Progress bar:** Visual representation of progress to next level
- **XP needed:** Shows how much XP is needed for the next level
- **Rank:** Shows the user's position in the server leaderboard

## Permissions Required
<Callout type="info">
- **View Channel:** Required to check level information
</Callout>

## Troubleshooting
<Callout type="warning">
- **No level data:** The user may not have earned any XP yet.<br />
- **XP not updating:** Make sure the XP system is enabled in the server.
</Callout>
