## Step-by-Step Example
1. Run the command:
   ```bash
   /sponsor
   ```
2. The bot will display information about supporting the bot's development to help keep it free and running for everyone, including:
   - Why support is needed
   - How to contribute
   - Interactive "Become a Sponsor" button

## Permissions Required
<Callout type="info">
- **No special permissions required** - Available to all users
</Callout>

## What You'll See
<Callout type="info">
Information about supporting the bot's development to help keep it free and running for everyone, including why support is needed, how to contribute, and an interactive "Become a Sponsor" button!
</Callout>

## Troubleshooting
<Callout type="warning">
- **No response:** Make sure the bot is online and has permission to send messages.<br />
- **Missing button:** The sponsor button should always appear with the command.
</Callout>
