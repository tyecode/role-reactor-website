## Parameters
<Cards>
  <Card title="limit (optional)" description="Number of users to show (1-20, default: 10)" />
  <Card title="type (optional)" description="Type of leaderboard: xp, level, messages, or voice (default: xp)" />
</Cards>

## Step-by-Step Example
1. To see the default leaderboard (top 10 by XP):
   ```bash
   /leaderboard
   ```
2. To see top 20 users by level:
   ```bash
   /leaderboard limit:20 type:level
   ```
3. To see top 5 users by messages:
   ```bash
   /leaderboard limit:5 type:messages
   ```
4. The bot will display the top users ranked by the selected type.

## Leaderboard Types
- **xp** - Total experience points (default)
- **level** - Current level
- **messages** - Total messages sent
- **voice** - Total voice time

## Features
- **Top rankings:** See the most active users
- **Multiple types:** Compare by XP, level, messages, or voice time
- **Customizable limit:** Show 1-20 users (default: 10)
- **Server insights:** Understand server activity patterns

## Permissions Required
<Callout type="info">
- **View Channel:** Required to view the leaderboard
</Callout>

## Troubleshooting
<Callout type="warning">
- **Empty leaderboard:** Users may not have earned XP yet.<br />
- **No data:** The selected leaderboard type may not have enough activity.
</Callout>

<Callout type="success">
- Use different leaderboard types to see various activity patterns (XP, level, messages, voice).<br />
