## Step-by-Step Example
1. To see your own avatar:
   ```bash
   /avatar
   ```
2. To see someone else's avatar:
   ```bash
   /avatar user:@username
   ```
3. The bot will display the avatar with interactive options.

## Features
- **Multiple sizes:** View avatar in different resolutions
- **Download options:** Get direct links to the avatar image
- **Format support:** Supports various image formats (PNG, JPG, WebP, GIF)
- **Animated avatars:** Shows animated avatars for users with GIF profile pictures

## Troubleshooting
<Callout type="warning">
- **User not found:** Make sure the user exists and is in the server.<br />
- **No avatar:** Some users may not have set a custom avatar.
</Callout>
