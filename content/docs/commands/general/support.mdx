## Step-by-Step Example
1. Execute the command:
   ```bash
   /support
   ```
2. The bot will display support information including:
   - How to get help
   - How to report issues
   - How to suggest features
   - Interactive buttons for Discord support server and GitHub repository

## Troubleshooting
<Callout type="warning">
- **No response:** Verify the bot is online and can send messages.<br />
- **Link not working:** Contact the bot developer for an updated invite.
</Callout>
