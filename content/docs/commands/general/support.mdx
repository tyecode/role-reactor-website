## Step-by-Step Example
1. Run the command:
    ```bash
    /support
    ```
2. The bot will reply with a link to the support server.

## Troubleshooting
<Callout type="warning">
- **No response:** Make sure the bot is online and can send messages.<br />
- **Link not working:** Contact the bot developer for an updated invite.
</Callout>
