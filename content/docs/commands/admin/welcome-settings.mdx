## Step-by-Step Example
1. Run the command:
   ```bash
   /welcome-settings
   ```
2. The bot will display:
   - Current welcome channel
   - Welcome message configuration
   - Auto-role settings
   - System status (enabled/disabled)
   - Embed format preference

## Information Displayed
- **Welcome Channel:** Which channel receives welcome messages
- **Welcome Message:** Current message template with placeholders
- **Auto-Role:** Role automatically assigned to new members
- **System Status:** Whether the welcome system is enabled
- **Embed Format:** Whether messages use embed format
- **Last Updated:** When settings were last modified

## Troubleshooting
<Callout type="warning">
- **Permission denied:** You need Manage Guild permissions to view welcome settings.<br />
- **No settings found:** Welcome system may not be configured yet.<br />
- **Settings not loading:** The bot may be experiencing issues.
</Callout>
