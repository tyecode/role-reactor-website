## Step-by-Step Example
1. Execute the command:
   ```bash
   /xp settings
   ```
2. The bot will display an interactive XP system management interface with buttons to:
   - Toggle the entire XP system on/off
   - Enable/disable message XP, command XP, role XP, and voice XP individually
   - View current XP amounts, cooldowns, and settings
   - Access configuration help and reset options

</Callout>

## Troubleshooting
<Callout type="warning">
- **Permission denied:** You require Manage Server permissions to configure XP settings.<br />
- **XP not working:** Verify if the XP system is enabled using the toggle button.<br />
- **No XP earned:** Users may be in cooldown or the specific XP type may be disabled.
</Callout>
