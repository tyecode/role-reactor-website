## Placeholders
You can use these placeholders in your goodbye message:
- `{user}` - User mention
- `{user.name}` - Username
- `{user.tag}` - User tag
- `{server}` - Server name
- `{memberCount}` - Total member count
- `{memberCount.ordinal}` - Ordinal member count (e.g., "1st", "2nd", "3rd")

## Troubleshooting
<Callout type="warning">
- **Permission denied:** You require Manage Server permissions to configure goodbye system.<br />
- **Channel not found:** Verify the channel exists and the bot can access it.<br />
- **Message not sending:** Verify bot permissions in the goodbye channel.<br />
- **Placeholders not working:** Ensure placeholders are spelled correctly (case-sensitive).
</Callout>
