## Placeholders
You can use these placeholders in your welcome message:
- `{user}` - User mention
- `{username}` - Username without mention
- `{server}` - Server name
- `{memberCount}` - Total member count
- `{userCount}` - User count (excluding bots)

## Troubleshooting
<Callout type="warning">
- **Permission denied:** You require Manage Server permissions to configure welcome system.<br />
- **Channel not found:** Verify the channel exists and the bot can access it.<br />
- **Auto-role not working:** Ensure the bot's role is positioned above the auto-role in the role hierarchy.<br />
- **Message not sending:** Verify bot permissions in the welcome channel.
</Callout>
