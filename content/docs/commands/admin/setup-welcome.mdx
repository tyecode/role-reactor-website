## Parameters
<Cards>
  <Card title="channel (required)" description="Channel where welcome messages will be sent" />
  <Card title="message (optional)" description="Welcome message (use placeholders like {user}, {server})" />
  <Card title="auto-role (optional)" description="Role to automatically assign to new members" />
  <Card title="enabled (optional)" description="Enable or disable the welcome system" />
  <Card title="embed (optional)" description="Use embed format for welcome messages" />
</Cards>

## Placeholders
You can use these placeholders in your welcome message:
- `{user}` - User mention
- `{username}` - Username without mention
- `{server}` - Server name
- `{memberCount}` - Total member count
- `{userCount}` - User count (excluding bots)

    ```bash
    /setup-welcome 
    channel:#welcome

    auto-role:@Member

    ```
  </Tab>

    Configure auto-role assignment:
    ```bash
    /setup-welcome 
    channel:#general 
    message:"Welcome {user}! You've been given the @Member role." 
    auto-role:@Member 
    enabled:true
    ```
  </Tab>
</Tabs>

## Permissions Required
<Callout type="warning">
- **Manage Guild:** Required to configure welcome system
- **Send Messages:** Required to post welcome messages
- **Manage Roles:** Required if using auto-role feature
- **Embed Links:** Required if using embed format
</Callout>

## Troubleshooting
<Callout type="warning">
- **Permission denied:** You need Manage Guild permissions to configure welcome system.<br />
- **Channel not found:** Make sure the channel exists and the bot can access it.<br />
- **Auto-role not working:** Ensure the bot's role is above the auto-role in the role hierarchy.<br />
- **Message not sending:** Check bot permissions in the welcome channel.
</Callout>
