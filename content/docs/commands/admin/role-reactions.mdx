## Permissions Required
<Callout type="warning">
- **Manage Roles:** Required to assign/remove roles from users
- **Send Messages:** Required to post the role message
- **Add Reactions:** Required to add reaction emojis
- **Embed Links:** Required to create rich embeds
</Callout>

## Troubleshooting
<Callout type="warning">
- **Bot not posting message:** Check bot permissions (Manage Roles, Send Messages, Add Reactions).<br />
- **Roles not assigned:** Ensure the bot's role is above the target roles in the Discord role list.<br />
- **Emoji not working:** Use standard emoji or custom emoji the bot has access to.<br />
- **Message not found:** Make sure the message ID is correct and the message still exists.
</Callout>

## Best Practices
<Callout type="success">
- Keep role lists manageable (5-12 per message is ideal).<br />
- Use clear, descriptive role names and appropriate emojis.<br />
- Test your setup with a test user before going live.<br />
- Regularly clean up unused role-reaction messages.
</Callout>
