## Implementation Guide

### Prerequisites

Before creating the role message, ensure the following roles exist in your server:

- **PC Gaming** - For PC gamers
- **PlayStation** - For PlayStation players
- **Xbox** - For Xbox players
- **Mobile** - For mobile gamers
- **FPS** - For first-person shooter fans
- **RPG** - For role-playing game enthusiasts
- **Racing** - For racing game fans
- **Strategy** - For strategy game players

## Troubleshooting

<Callout type="warning">
- **Roles not assigned:** Verify bot permissions and role hierarchy order
- **Emoji not working:** Ensure emojis are accessible to the bot
- **Members confused:** Pin a message explaining each role's purpose
</Callout>

## Best Practices

<Callout type="success">
- Organize roles by platform and game genre for clarity<br />
- Use clear, recognizable emojis for quick identification<br />
- Regularly update role lists to match community preferences<br />
- Create separate messages for different role categories
